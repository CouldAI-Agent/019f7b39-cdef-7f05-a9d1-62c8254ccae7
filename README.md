# Investment Tracker

A modern, easy-to-use cross-platform application designed for individuals to monitor and grow their personal investments. Built with Flutter, it features a clean, minimalist layout with a professional color palette (green, navy, black, and white).

## Features
- **Portfolio Dashboard**: High-level overview of total portfolio value and overall returns.
- **Asset Allocation**: Visual breakdown of your investments using an interactive pie chart.
- **Individual Holdings**: Detailed list of assets (stocks, ETFs, crypto) with current value, shares, and profit/loss indicators.
- **Minimalist Design**: A clean UI focusing on readability and professional aesthetics, perfectly responsive across mobile, tablet, and desktop devices.

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart
- **Charting**: `fl_chart` for data visualization

## Getting Started

1. Ensure you have Flutter installed (version 3.7.2 or higher).
2. Clone this repository and navigate to the project directory.
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
